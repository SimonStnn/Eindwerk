Waarvoor wordt het gebruikt

Dit is het brein van de operatie. Hier gaan alle ESP's hun data naartoe sturen. Ook de client verbind via websocket naar de raspberry pi.

Het heeft een webserver waar je naartoe kan verbinden. Daarop vind je de live locaties van de toestellen. Om dit mogelijk te maken word een websocket verbinding gemaakt. Hieronder vind je een tabel met de verschillende commandos die verstuurd worden.

Commando Beschrijving Antwoord Beschrijving
REQ=collection Client vraagt collection op. COLLECTION={collection} Collection in json format
UPDATE_POS= Update de positie van een satelliet.
CHANGE_ROOM= Verander de room waar de satelliet zich bevind.
RENAME= Verander de naam van een satelliet.
DEBUG= Verander het debug level van een logger.
